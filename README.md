# CRUD-api

## Running


1. Clone repository

```bash
git clone https://github.com/RVitaly1978/crud-api.git
```

2. Checkout to `develop` branch

```bash
git checkout develop
```

3. Install dependencies

```bash
npm install
```

4. Run program

```bash
# in develop node
npm run start:dev

# in production node
npm run start:prod
```
